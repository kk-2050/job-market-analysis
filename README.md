# LinkedIn Job Market Analysis Dashboard

## Project Overview
Analyzed 123,000+ LinkedIn job postings to uncover job market trends, salary insights, and in-demand skills using an automated data pipeline.

## Motivation
Built this project during my own job search to gain data-driven insights into the job market — making the analysis both authentic and personally relevant.

## Tech Stack
| Tool | Purpose |
|------|---------|
| Python | Data cleaning, automation |
| SQL (SQLite) | Data aggregation and querying |
| Claude API | AI-powered insight generation |
| Power BI | Interactive dashboard |
| R (ggplot2, dplyr) | Statistical analysis and visualization |
| Kaggle API | Automated data retrieval |
| GitHub | Version control and portfolio publishing |

## Key Findings
- **Mid-Senior level** roles are most in demand (41,489 postings)
- **76% of job postings** did not disclose salary information
- **Executive roles** earn 4.8x more than Entry level on average
- **Chicago, IL** ranks 3rd nationally for job opportunities
- **Full-time** roles dominate the market

## Pipeline Architecture
Kaggle API → Python → SQL → Claude API → Power BI → R → GitHub
